## 🚀 Quick Start

### Prerequisites

- **Node.js** 18+
- **npm** or **pnpm**
- API keys for: Supabase, Groq, RapidAPI (JSearch), Hunter.io

### 1. Clone & Install

```bash
git clone https://github.com/Ashikvk18/hirelens.git
cd hirelens
npm install
```

### 2. Environment Variables

Create `.env.local` in the project root:

```env
# Supabase
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key

# Groq AI
GROQ_API_KEY=your_groq_api_key

# JSearch (RapidAPI)
RAPIDAPI_KEY=your_rapidapi_key

# Hunter.io
HUNTER_API_KEY=your_hunter_api_key
```

### 3. Database Setup

Run the schema in your Supabase SQL editor:

```bash
# Copy the contents of supabase/schema.sql into the Supabase SQL Editor
```

### 4. Run Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) — you're live! 🎉

## 🤖 AI Features Deep Dive

### Resume Analysis Pipeline
```
Resume + Job Description
        │
        ▼
┌───────────────────┐
│  Keyword Matching │ → Missing keywords, match %
│  Section Analysis │ → Weak sections identified
│  Risk Assessment  │ → Rejection probability
└───────┬───────────┘
        │
        ▼
┌───────────────────┐
│  AI Suggestions   │ → 5 actionable improvements
│  AI Rewriter      │ → 3 rewritten resume versions
│  Outreach Gen     │ → LinkedIn + email messages
└───────────────────┘
```

### Cover Letter Generation
- **Input**: Resume + Job Description + Tone preference
- **Tones**: Professional · Confident · Enthusiastic
- **Output**: 250-350 word cover letter + key highlights used
- **Features**: Copy to clipboard, download as .txt, regenerate

### Interview Prep
- **Technical**: Data structures, system design, coding challenges
- **Behavioral**: STAR method, situational scenarios, culture fit
- **Personalized**: Questions tailored to specific role + company

## 🎯 Motion Design System

HireLens uses a comprehensive shared animation library (`src/lib/motion.ts`):

| Variant | Effect | Use Case |
|---------|--------|----------|
| `scrollFadeUp` | Fade + slide up 24px | General sections |
| `scrollBlurUp` | Fade + slide up + blur | Premium headings |
| `scrollScaleIn` | Fade + scale from 0.92 | Cards, CTAs |
| `scrollFadeLeft/Right` | Directional slide | Alternating layouts |
| `scrollStagger` | Parent stagger container | Card grids, lists |
| `scrollStaggerItem` | Child fade + slide | Individual items |
| `scrollStaggerItemBlur` | Child fade + slide + blur | Premium items |

**Mouse Parallax Hook** (`useMouseParallax`):
- 3 depth layers: shallow (±12px), mid (±30px), deep (±50px)
- Scene tilt: ±8° rotation toward cursor
- Spring physics: stiffness 50, damping 30
